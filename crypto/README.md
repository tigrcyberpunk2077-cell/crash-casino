# $RAMIN — мем-токен на Solana 🐏

Тулкит для выпуска SPL-токена с метаданными (имя, тикер, лого) через Metaplex.

## Что делает `launch.mjs`
1. Заливает лого и JSON-метаданные на Arweave (через Irys).
2. Создаёт mint + on-chain метаданные.
3. Чеканит всю эмиссию (1 млрд) на твой кошелёк.
4. Отзывает mint authority (эмиссию нельзя увеличить) и freeze authority (нельзя морозить кошельки) — честный мем-вайб.

## Установка
```bash
cd crypto
npm install
```

## Шаг 1. Кошелёк
```bash
npm run keygen          # создаст keypair.json (приватный ключ — НЕ коммить!)
```
`keypair.json` совместим с импортом в Phantom/Solana CLI.

## Шаг 2. Тест на devnet (бесплатно, рекомендую сначала)
```bash
# пополни devnet-кошелёк: https://faucet.solana.com (вставь адрес из keygen)
NETWORK=devnet npm run balance
NETWORK=devnet npm run launch
```

## Шаг 3. Боевой запуск на mainnet
1. Переведи на адрес кошелька **~0.05 SOL** (газ + загрузка метаданных).
2. Запусти:
```bash
NETWORK=mainnet npm run launch
```
Скрипт даст 5 сек на отмену (Ctrl+C) перед тратой реального SOL.

## Настройка
Всё редактируется в `config.mjs`: имя, тикер, описание, лого, decimals, эмиссия, отзыв прав.
Лого по умолчанию — `assets/logo.svg`. Для кошельков лучше заменить на **PNG 512×512**
(`imageFile: "assets/logo.png"`).

## После выпуска
Чтобы токеном можно было торговать — создай пул ликвидности на Raydium/Orca
(пара RAMIN/SOL). Это отдельный шаг и реальные деньги в пул.

## ⚠️ Дисклеймер
Это шуточный мем-токен. Не стейблкоин, не инвестиция, не финсовет.
Приватный ключ (`keypair.json`) — единственный доступ к токенам и кошельку. Не теряй и не показывай его.
